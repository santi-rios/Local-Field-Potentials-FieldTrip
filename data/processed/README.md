# Processed Data Directory

This directory contains preprocessed LFP data.

## Contents
- Filtered and cleaned LFP data
- Data structures in FieldTrip format
- Preprocessed files ready for analysis

## Important Notes
- Processed files are excluded from version control (see .gitignore)
- Files are typically saved as .mat files
- Keep track of preprocessing parameters used

## Typical Files
- `preprocessed_data.mat` - Main preprocessed dataset
- `*_cleaned.mat` - Individual session files after cleaning
