# Raw Data Directory

Place your raw LFP recording files here.

## Supported Formats
- Neuralynx (.ncs)
- Plexon (.nex)
- Blackrock (.nev)
- MATLAB (.mat)

## Important Notes
- Large data files are excluded from version control (see .gitignore)
- Organize files by session or subject as needed
- Keep backup copies of raw data in a secure location

## Example Structure
```
raw/
├── subject01_session01.ncs
├── subject01_session02.ncs
├── subject02_session01.ncs
└── ...
```
