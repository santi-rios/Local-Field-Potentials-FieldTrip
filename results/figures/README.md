# Results - Figures

This directory contains generated figures from the analysis pipeline.

## Figure Types
- Power spectral density plots
- Time-frequency representations
- LFP trace visualizations
- Statistical comparison plots
- Coherence plots

## Formats
Default format is PNG, but can be configured in config.m:
- PNG (web-friendly)
- JPEG (compressed)
- FIG (MATLAB figure, editable)
- EPS (publication quality)

## Important Notes
- Most figure files are excluded from version control
- Keep figures organized by analysis type or date
- Consider creating subdirectories for different experiments
